# ⏰ Month 11 — Time Series & Spark at Scale

**Duration:** 100 hours
**Difficulty:** ⭐⭐⭐⭐ Very Hard
**Goal:** Forecast the future and scale processing to big data

## Sprints

### Sprint 1: Time Series Fundamentals (Days 1–10)
- Time series components (trend, seasonality, residuals)
- Stationarity and differencing
- Autocorrelation (ACF) and partial autocorrelation (PACF)
- Time series visualization

### Sprint 2: ARIMA & Prophet (Days 11–24)
- ARIMA (AutoRegressive Integrated Moving Average)
- Model selection and parameter tuning
- Facebook Prophet for intuitive forecasting
- Handling seasonality and trends

### Sprint 3: Cloud & AWS Basics (Days 25–32)
- AWS S3 bucket operations
- EC2 instance basics
- IAM and security
- Lambda functions for data processing

### Sprint 4: Spark Essentials (Days 33–47)
- Spark DataFrames
- RDDs basics
- Spark SQL
- MLlib for distributed ML

### Sprint 5: End-to-End Pipeline (Days 48–50)
- Building scalable pipelines
- Cloud deployment
- Monitoring and performance

## Project: Sales Forecasting on Cloud

**Objective:** Build a scalable forecasting system on AWS
**Skills:** Time series, Spark, cloud deployment
**Deliverables:**
- Historical data in S3
- Spark preprocessing job
- ARIMA/Prophet models
- Forecasts
- Dashboard or API
- Cost analysis
